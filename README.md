# Alien invasion

![img.png](app/images/img.png)

Steps for playing:

1. Get the project

```bash
git@github.com:Yurii-Solodzhuk/alien_invasion.git
cd alien_invasion
```

2. Configure venv

```bash
virtualenv -p python3 .venv
```

3. Install requirements

```bash
pip install -r requirements.txt
```

4. Enjoy the game

```bash
python main.py
```
